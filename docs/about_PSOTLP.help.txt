TOPIC
    about_PSOTLP

SHORT DESCRIPTION
    PSOTLP emits OpenTelemetry Protocol (OTLP) logs, traces, and metrics
    from PowerShell. It ships with cmdlets for connection management,
    structured logging, hosted script execution with stream capture, and
    span/trace tracking.

LONG DESCRIPTION
    PSOTLP is a C# binary PowerShell module targeting .NET Standard 2.0.
    It supports both Windows PowerShell 5.1 and PowerShell 7+. Telemetry is
    sent synchronously over OTLP/HTTP (JSON, Protobuf, or NDJSON). All
    authentication headers are supplied through the single -Headers
    IDictionary on Connect-OTLP; values are stored as SecureString and only
    materialized at the request boundary.

    Typical workflow:

        $bearer = ConvertTo-SecureString -String 'Bearer eyJhbGciOi...' -AsPlainText -Force
        Connect-OTLP -EndpointUri 'https://otel.example.com' `
            -Headers @{ Authorization = $bearer } -ServiceName 'powershell-session'

        Write-OTLPLog -Body 'Starting device onboarding' -Severity Information

        Invoke-OTLPScript -ScriptBlock {
            Get-Service | Select-Object -First 5
        }

        Disconnect-OTLP

    Span/trace workflow:

        $span = Start-OTLPSpan -Name 'install-driver' -Kind Internal
        Write-OTLPSpanEvent -SpanId $span.SpanId -Name 'driver-located'
        Stop-OTLPSpan -SpanId $span.SpanId

CMDLETS
    Connect-OTLP            Open a connection to an OTLP endpoint.
    Disconnect-OTLP         Close the current OTLP connection.
    Get-OTLPConnection      Return the active connection view.
    Write-OTLPLog           Send a single log record.
    Send-OTLPLogBatch       Send a batch of log records.
    Invoke-OTLPScript       Execute a script block in a hosted runspace
                            with stream capture; supports -ImportFunctions,
                            -ImportVariables, and -SharedState for parent
                            scope sharing.
    Start-OTLPSpan          Start an OTLP span.
    Stop-OTLPSpan           Stop an OTLP span.
    Write-OTLPSpanEvent     Add an event to an active span.
    Send-OTLPTraceBatch     Send a batch of spans.
    Write-OTLPMetric        Emit a single Gauge or Sum metric data point.
    Send-OTLPMetricBatch    Send a batch of metric data points.

ENVIRONMENT VARIABLES
    PSOTLP_OTLP_ENDPOINT           Base OTLP endpoint.
    OTEL_EXPORTER_OTLP_ENDPOINT    Standard OTel fallback for the endpoint.
    PSOTLP_SERVICE_NAME            Default service.name resource attribute.
    OTEL_SERVICE_NAME              Standard OTel fallback for service name.

SECURITY
    All header values are treated as sensitive and stored as SecureString.
    Header values are never written to internal log lines or telemetry
    payloads. The redaction engine scrubs Authorization headers, bearer
    tokens, api-key key/value pairs, password-like values, and PEM private
    key blocks from captured script content.
